# basic terminal Commands

### Show all list file
`ls`

### show working directory ( where am i )
`pwd`

### clear screen
`clear`

### Change directory
`cd`

- **for one step:**
`cd desktop`
- **for multi step:** 
`cd desktop/Reacting`

### Go back command
- **for one step:**
`cd..`
- **for multi step:** 
`cd ../..`

## Absolute and Relative Path
- **Relative Path:**
`cd desktop/Delta`
- **Absolute Path**
`cd /onedrive/desktop/Delta`

### Root directory
`/`

### Home directory
`~`

---

### Make directory ( Create folder )
`mkdir`

## Touch Commands
Used to create files

`touch index.html`      
`touch app.js`      
`touch abc.txt`     
`touch style.css`       

### Deleting File and Folders
`rm`

- **removes files:**
`rm abc.txt`

- **removes empty folders:**
`rmdir`

- **removes any folders:**
`rm-rf`
